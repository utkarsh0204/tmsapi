## Task1
* minimum php8.2 required
* cd your-project-name
* copy .env.sample and rename as .env file
* update the db details in .env

* run below at root folder
 * php artisan key:generate
 * php artisan migrate
 * php artisan db:seed
 * php artisan serve

* api server will be started
